1. システム概要

プログラミングスクール生徒名簿管理システムは、生徒情報を効率的に管理するためのWebアプリケーションです。

  • 生徒情報の登録・編集・削除
  • リアルタイム検索機能
  • 性別によるフィルタリング
  • CSVファイルへのエクスポート機能
  • 統計情報の自動表示

2. 基本操作

生徒の追加

  1. 画面上部の「+ 生徒を追加」ボタンをクリック
  2. 表示されるフォームに以下の情報を入力
    • 氏名（必須）
    • ふりがな（必須）
    • 生年月日（必須）
    • 性別（必須）
    • 得意分野（必須）
  3. 「保存」ボタンをクリックして登録完了

生徒情報の編集

  1. 編集したい生徒の行にある「編集」ボタンをクリック
  2. 表示されるフォームで情報を修正
  3. 「保存」ボタンをクリックして変更を適用

生徒の削除

注意: 削除操作は取り消せません。削除前に必ず確認してください。
  1. 削除したい生徒の行にある「削除」ボタンをクリック
  2. 確認ダイアログで「OK」をクリック

3. 検索・フィルター機能

検索機能

検索ボックスに以下の情報を入力すると、リアルタイムで検索結果が表示されます：

  • 氏名（部分一致）
  • ふりがな（部分一致）
  • 得意分野（部分一致）

フィルター機能

性別でフィルタリングが可能です：

  • 全て: すべての生徒を表示
  • 男子: 男性の生徒のみ表示
  • 女子: 女性の生徒のみ表示

4. データのエクスポート

生徒データはCSV形式でエクスポートできます。

ヒント: エクスポートされたCSVファイルは、ExcelやGoogleスプレッドシートで開くことができます。

エクスポート手順

  1. ブラウザの開発者ツールを開く（F12キー）
  2. コンソールタブを選択
  3. 以下のコマンドを入力：
    downloadCSV()
  4. Enterキーを押すとCSVファイルがダウンロードされます

5. キーボードショートカット

操作 ショートカット
検索ボックスにフォーカス Ctrl + F (Windows) / Cmd + F (Mac)
モーダルを閉じる Esc

6. トラブルシューティング

データが表示されない場合

  • ブラウザのキャッシュをクリアして再読み込み
  • JavaScriptが有効になっているか確認
  • コンソールにエラーが表示されていないか確認

保存できない場合

  • 必須項目がすべて入力されているか確認
  • 生年月日の形式が正しいか確認

7. 推奨環境

  • Chrome 90以降
  • Firefox 88以降
  • Safari 14以降
  • Edge 90以降
注: Internet Explorerはサポートされていません。
